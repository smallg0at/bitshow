炽焰咆哮虎为什么是神（转自B站某整活区up主）

       炽焰咆哮虎为什么是神，在讨论这个问题之前，我想先说说其他宝可梦相较于咆哮虎究竟差在哪里？

       首先是犯下傲慢之罪的巨碳山，让队友帮他触发弱点保险后就敢上升十段能力，这种傲慢的宝可梦注定走不远。事实也是如此，靠着超极巨化三回合变态的输出能力在players cup拿了一堆名次后一直在走下坡路，最后泯然众人（指使用率）。与之相比，神就很谦卑，八代失去了本系拍落，各种新增建议威吓效果的特性都是神留给众人的慈悲。神不是不能极巨化输出，只是不想通过太完美的表现让对手绝望所以可以打辅助。可笑有些人不了解神的良苦用心，居然说出“咆哮虎只是个队伍润滑剂算什么宇宙第一”这种话，我劝你们好自为之。

       然后是犯下愤怒之罪的恶系武道熊师，因为对面宝可梦弱恶就无视保护和威吓，极巨化两倍血量穿死，为了达成这种夸张的效果选择抛弃保命的气腰选择了专头，违背了神在新约中的一句话“威吓一次以后你的伤害也不过如此”。于是神降下了他的惩罚，花疗环环的吸取之吻，仙布的电光一闪，电柱子的大威力电本都将他打的一蹶不振。

       接着是犯下懒惰之罪的卡璞鳍鳍，自以为拿下两次wcs冠军就了不起，自创冥想强化法。另外大家有所不知，其实在vgc2017的wcs赛前，卡璞鳍鳍闭上眼睛做冥想的时候，其脑海中看到的画面，正是将在vgc2018利用大声咆哮反复削弱特攻手的咆哮虎。那时神告诉他，“你的使用率只可超过土地云，不可越过”。然而，神的劝说不但没有让卡璞鳍鳍迷途知返，他竟然还敢在威吓解禁前夕说出“两猫相争，鳍鳍得利”这种话正面挑战神的威严。于是神降下了他的惩罚，卡璞鳍鳍在第八世代因为青草滑梯身败名裂，两届wcs冠军没人记得，从此沦为众人队中只会吃剩饭的要塞。

       再然后是犯下嫉妒之罪的轰擂金刚猩，屡次被神的闪焰冲锋击败，但口服心不服的卡璞猩猩，竟然敢当众说出85速击掌奇袭拍住神，感觉自己在拍人方面更强这种话。于是，在g8目前含金量最高的比赛，即players cup上，在半决赛便被神吓的不敢出场，输掉了比赛。不过，念在猩弟最终皈依了神的教，并且践行了神的意志用青草滑梯讨伐了屡次以下犯上的卡璞鳍鳍，神应许了他在使用率排行榜上的一切。

       犯下贪婪之罪的雷吉艾勒奇，只是初出茅庐，雷吉艾勒奇就拿下了使用率第一。而这，正是因为神的应允，神本来钦定他为自己在第八世代的代言人，在自己退场之后作为救火队长替自己讨伐雨天队的盖欧卡，蚊香蛙皇等魔头。甚至不惜亲自为电柱子铺路，用抛下狠话主动退场，将一切荣耀归于他，甚至应允他可以待在神长期占据的使用率第一的位置。但最后雷吉艾勒奇却被力度蒙蔽了双眼，不再以践行神的理想为自己的理想，也不再聆听神的教诲，甚至自以为地位已经超过了神。竟然让拥簇者组出“眼镜十万秒了，就这”这种亵渎神尊严的配置。于是神降下了惩罚，雷吉艾勒奇从此没有任何补盲技能，并且被雷丘，阿罗拉地区嘎啦嘎啦等宝可梦吃死，而他的十万伏特再也不能秒杀盖欧卡，他的电网再也不能先手压制两人，他的队友再也不能凭借免疫放电而狐假虎威，他的队友再也不能稳定的给他掩护，阻止开空间。

       犯下暴食之罪的伊裴尔塔尔，在神子雷吉艾勒奇堕落之后，神开始寻找下一位代言人。这一次，他相中了作为一级神的y鸟。相较于神，伊裴尔塔尔无疑是不完美的。他没有神以恶系之身抵御哲尔尼亚斯月亮之力的火属性，也没有神深不见底的技能池，不占神位、随时退场的余裕，无懈可击的联防地位。但神将自己最有代表性，同时也是影响力最大的招式【大声咆哮】暂时交给了他，并让他可以用气场强化大声咆哮的威力，但由于冠之雪原的事，神认为或许是自己特耐太高导致了神子的堕落，于是神这次选择将突击背心让给了伊裴尔塔尔，降低自己对抗特攻手的综合耐久。最终，伊裴尔塔尔顶住了月亮之力。然而，撑下一击的伊裴尔塔尔不仅冒大不韪在暗黑气场下使用本系欺诈，更妄图染指神在队伍中的恶系位置。只是他错误的认为神的力量来自第七世代可以回一半血的巨树果。最终他也因此犯下了暴食之罪，陷入了混乱状态。

       最后，犯下淫欲之罪的败露球菇。冠之雪原之后，神的综合耐久已经很吃紧了，邪神盖欧卡用雨天喷水或极巨水流清空了神的血量槽。神已经无力再惩罚y鸟，只有信徒依旧在维护神的威严而已。此刻已经决定藏在后排的神在最后依然心系对战。于是他选中了第三位神子——棱瓜果盾菇。作为第三位神子，抗飞果盾菇很好的维护了神的威严，但在连续催眠两代堕落神子以后，盾菇遇到了雌性爱管侍。被精神场地广域战力接连打死，于是在狼狈的带上气腰，最终因为无法同时对抗极巨飞冲和广域战力之后，使用率排名一落千丈。

       虎门

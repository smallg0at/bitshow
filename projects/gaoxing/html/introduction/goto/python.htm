大数定律

问题背景

小附同学想要验证大数定律，但是他懒得自己实验，而且不会编程，现在希望你可以帮助小附同学，用直观的方式展示大数定律

实施想法

用random函数随机坐标，海龟画图画出相应位置，后根据平面直角坐标系四象限判断位置域，算出频率，对比基本概率

                
    import turtle;import time;import random
    i = int(input())
    x = 0
    turtle.setup(300,300)
    turtle.speed(50)
    turtle.goto(300,0)
    turtle.goto(-300,0)
    turtle.goto(0,0)
    turtle.goto(0,300)
    turtle.goto(0,-300)
    turtle.goto(-300,-300)
    turtle.goto(-300,300)
    turtle.goto(300,300)
    turtle.goto(300,-300)
    turtle.goto(-300,-300)
    x1 = 0;x2 = 0;x3 = 0;x4 = 0
    while True:
        if x == i:
            break
        x = x + 1
        a = float((random.randint(-300,300)));b = float((random.randint(-300,300)))
        if a<0 and b>0:
            x2=x2+1
        if a>0 and b>0:
            x1=x1+1    
        if a<0 and b<0:
            x3=x3+1
        if a>0 and b<0:
            x4=x4+1    
        turtle.penup()
        turtle.goto(a,b)
        turtle.pendown()
        turtle.circle(1)
        turtle.penup()
        turtle.goto(0,0)
        turtle.pendown()
    time.sleep(3)
    print("总共："+str(i)+"，第一象限："+str(x1)+"，第二象限："+str(x2)+"，第三象限："+str(x3)+"，第四象限："+str(x4))
    print("第一象限概率："+str(round((x1/i),5))+"，第二象限概率："+str(round((x2/i),5))+"，第三象限概率："+str(round((x3/i),5))+"，第四概率概率："+str(round((x4/i),5)))

数据量越大，频率越接近概率

某道概统的数学题

题目要求

已知一组数据 1，2，3，4，5，6，7，8，9. 其平均值为a=5;标准差为s=2*(15**0.5)/3.现从中随机抽取 3 个数据作为样本(共有 84 种不同的方法)共 10 次，记其样本平均数为x,d = |x - a|, 估计d < s的概率和d < 2s的概率.（可以用电脑编程帮助完成）

实施想法

按题目要求实施

初:用random函数随机，得到数据之后判断相等，设定两个变量分别代表两事件的频数，发生一次加一，最后频数除以总数得到结果

问题:每次频率相差比较大，没有清楚的普遍规律

改:使用嵌套循环，使频率更接近概率

                
    import random
    m=[];n=[]
    s=(20/3)**0.5
    m0=0;n0=0
    for i in range(10000):
        m0=0;n0=0
        for j in range(10):
            while True:
                a=random.randint(1, 9);b=random.randint(1, 9);c=random.randint(1, 9)
                if a!=b and a!=c and b!=c:
                    break
            x=(a+b+c)/3
            d=((x-5)**2)**0.5
            if d < s:
                m0=m0+1
            if d < 2*s:
                n0=n0+1
        m.append(m0);n.append(n0)
    a=0;b=0
    for i in range(len(m)):
        a=a+m[i];b=b+n[i]
    print(a/1000);print(b/1000)
